# API (backend)

Descrição

Backend em Python que expõe endpoints para predizer/consultar dados relacionados à osteoporose.

Arquivos importantes

- `api/main.py` — ponto de entrada principal
- `api/app/main.py` — possível aplicação interna

Como rodar (ex.: PowerShell)

```
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt  # se existir em api/
python -m api.main
```
